## What is PromptAmp?

PromptAmp is an open-source, cross-browser extension (Chrome · Firefox · Edge) that rewrites your rough prompt drafts into well-engineered prompts — right inside the input field you're already typing in. ChatGPT, Gemini, Claude, Midjourney, image & video generators, anywhere.

### Why it's different

- 🔑 **BYOK — Bring Your Own Key.** Your API key is stored locally in your browser and sent only to the provider you choose. No middleman server, no subscription, no backend — ever.
- 🌐 **Universal.** Works on any editable text field on any site — not a whitelist of chat apps.
- 🎛️ **Profiles.** Domain-aware enhancement: image prompts ≠ code prompts ≠ study prompts. Polished built-in profiles, fully customizable.
- 🔒 **Private by design.** No servers, no telemetry, no accounts. The code is public so you can verify every claim.

### Planned provider support

OpenAI · Anthropic · Google Gemini · OpenRouter (one-click connect) · Groq · Ollama / LM Studio (local)
